# Frontend Team Project "Меблерія"

Інтерфейс для онлайн-магазину меблів з пошуком, відгуками та інтеграцією з
бекендом.

## Можливості

- Пошук меблів за назвою, категорією або ключовими словами
- Відображення відгуків користувачів з рейтингами
- Автоматичний розрахунок рейтингу на основі середнього балу
- Інтеграція з бекендом через REST API
- Адаптивний дизайн для мобільних і десктопних пристроїв

## Технології

- Vite
- Axios для HTTP запитів
- HTML5
- CSS3
- JavaScript

### Запуск проекту

- Клонування репозиторію
- Встановлення залежностей `npm install`
- Запуск у режимі розробки `npm run dev`
